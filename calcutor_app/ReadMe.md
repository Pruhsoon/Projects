Steps To Run this on Your machine:

Step 1: Clone this repo-->
        gh repo clone Pruhsoon/Projects

Step 2: Install Nodemon-->
        npm i nodemon

Step 3: Run command -->
        nodemon server.js

Step 4: Open your browser and insert URL --> localhost:3000
